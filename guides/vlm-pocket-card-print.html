Receiving Deliveries
  1. Request BOL and/or delivery note from driver.
  2. Visual inspection of delivery against BOL.
  3. Verify contents against packing list.
  4. Check for damages or missing items.
  5. Note damages/missing on all copies of BOL.
  6. Inform Materials and LCC.
  7. Send BOL + packing list to Materials and LCC by end of day.
Materials closes the transaction in SM²ARTA.
VLM says: No BOL = it never happened. Call LCC before you touch anything. The driver can’t leave until you’ve counted.

Confirming Deliveries + Assembly

Delivery Confirmation • Assembly • SM²ARTA
Arrives
  1. 1Match CORT Delivery List to what’s on the truck — item by item.
  2. 2CORT staff installs — supervise, count, verify placement.
  3. 3Sign only when every item is in place. Discrepancy → note it.
  4. 4Report to Materials: received / partial / issues.
Arrives
  1. 1Check BOL ref: FWC26 LOG / LSR-### — your shipment ID.
  2. 2Count pallets vs BOL before driver leaves.
  3. 3Open, inspect, match packing list.
  4. 4Report to Materials: received / not received / OS&D filed.
Arrives
  1. 1Pallet or parcel (FedEx/UPS)? Check tracking if courier.
  2. 2Items pre-sorted by FA — match to order file.
  3. 3Open 3 random kits, verify against order file. Drop each to requestor.
  4. 4Report to Materials: received / not received / discrepancies.
After EVERY delivery: Confirm to Materials & LCC • Send BOL + packing list • Report issues — same day
